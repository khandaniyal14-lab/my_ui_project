If the email verification link didn't work (common on mobile devices), you can verify your email manually using the form below.

How to find your verification token:

  1. Check your email from Africa House Pakistan
  2. Look for the section that says "verification token"
  3. Copy the token (it's a long string of letters and numbers)
  4. Paste it in the field above

Still having trouble?

If you can't find the verification token or this doesn't work:

  1. Try registering again with the same email
  2. Make sure you're using the same device/browser
  3. Check your spam/junk folder for the email
  4. Contact support if the problem persists
